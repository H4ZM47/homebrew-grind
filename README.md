# H4ZM47 Task

## How do I install these formulae?

`brew install h4zm47/task/<formula>`

Or `brew tap h4zm47/task` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "h4zm47/task"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
