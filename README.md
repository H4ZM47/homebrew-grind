# H4ZM47 Grind

## How do I install these formulae?

`brew install h4zm47/grind/<formula>`

Or `brew tap h4zm47/grind` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "h4zm47/grind"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
